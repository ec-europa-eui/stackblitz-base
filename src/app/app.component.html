isLoading

Action buttons with a loading state visual is used in some cases to improve better user experience when an action takes too long time to complete. Use the isLoading input option with true | false value to enable/disable it.

Button link

As a backward compatible feature, you can also style a text anchor link like a button by using the following classes :

  • - for internal links: class="eui-button eui-u-text-link"
  • - for external links: class="eui-button eui-u-text-link-external"
